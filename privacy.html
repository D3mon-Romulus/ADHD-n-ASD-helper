Our Commitment to Privacy

We are committed to protecting the privacy of children and families using this application. This policy explains our data practices.

Information We Do NOT Collect

This application does NOT collect, transmit, or store any personal information on external servers. We do not:

  • Collect names, emails, or contact information
  • Track user behavior or analytics
  • Use cookies for tracking purposes
  • Share data with third parties
  • Sell or monetize user data
  • Require user accounts or registration

Local Data Storage

All application data is stored locally on your device using browser localStorage:

  • Child profiles (names and ages)
  • Tasks and routines
  • Behavior tracking records
  • Points and progress data
  • Application settings and preferences

This data never leaves your device and is only accessible on the device where it was created.

Data Control

You have complete control over your data:

  • You can delete all data at any time through the Parent Dashboard
  • Clearing browser data will remove all stored information
  • No data recovery is possible once deleted
  • You are responsible for backing up data if needed

Children's Privacy (COPPA Compliance)

This application is designed for children aged 3-18 but does not collect any personal information as defined by COPPA (Children's Online Privacy Protection Act). Since no data is transmitted or stored on external servers, traditional COPPA consent mechanisms are not required. However:

  • Parental supervision is strongly recommended
  • Parents should review all features before allowing child use
  • The Parent PIN system provides basic access control

Third-Party Services

If you install this as a PWA or access it through a hosting platform:

  • The hosting provider may collect basic server logs (IP addresses, access times)
  • These logs are not controlled by this application
  • Refer to your hosting provider's privacy policy for details

Security

Since all data is stored locally:

  • Security depends on your device's security measures
  • Use device passwords/locks to protect data
  • Be cautious when using on shared devices
  • The Parent PIN provides basic app-level protection

Changes to Privacy Policy

Any changes to this policy will be posted here. Significant changes will be noted in the application.
